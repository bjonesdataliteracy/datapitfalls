datapitfalls → Semiotic annotation bridge

A chart was scanned by datapitfalls; each finding became a Semiotic v3 annotation spec via buildSemioticAnnotationBridge(). The bridge emits them unanchored (no x/y), so this host app positions them — here, anchored to bars in the Semiotic frame and listed in the margin. (On v3 an unanchored annotation is dropped until the host resolves its coordinates, so anchoring is required, not optional.)
