# Code of Conduct

We're committed to making participation in the datapitfalls community a welcoming, respectful, and harassment-free experience for everyone.

This project adopts the **Contributor Covenant, version 2.1**. You can read the full text here:

📜 **https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

## Enforcement

If you experience or witness unacceptable behavior, please report it confidentially to the project maintainers at **bjones@dataliteracy.com**. All reports will be reviewed and investigated promptly and fairly.

By participating in this project, you agree to abide by the Contributor Covenant. Thanks for helping keep this community kind and constructive.
